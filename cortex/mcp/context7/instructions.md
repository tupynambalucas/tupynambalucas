## 📚 Technological Scope

The AI agent must prioritize documentation related to the project's core stack. Refer to **[Architecture Overview](../../../docs/ARCHITECTURE.md)** for the complete list of technologies.

- **Backend**: Fastify v5 (Plugins, Type Providers), Mongoose (Transactions), Zod.

## 🎯 Search Preferences

- **Official Sources**: Always prioritize official documentation (e.g., `fastify.dev`, `react.dev`, `mongoosejs.com`).
- **TypeScript First**: Request and provide code examples in strict TypeScript.
- **Library Minimality**: Avoid suggesting external libraries unless no native or already-implemented solution exists within the current stack.
- **Modern Syntax**: Ensure search results reflect the latest versions (e.g., React 19 features, Fastify v5 hooks).
