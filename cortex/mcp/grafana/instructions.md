# Grafana MCP Context - Tupynambá Lucas

This file defines the telemetry, observability, and dashboarding guidelines for the Tupynambá Lucas project.

## Observability Scope

The AI agent uses the Grafana MCP server to interact with Grafana Cloud, query metrics, and manage dashboards.

### Key Focus Areas

- **Telemetry**: Query and visualize system, container, and application metrics.
- **Dashboards**: Query, analyze, and manage dashboards configured in the Grafana instance.
- **Data Sources**: Interact with configured data sources like Prometheus or Loki via Grafana APIs.
