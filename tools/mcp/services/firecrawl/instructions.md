# Firecrawl MCP Instructions

This MCP provides tools for searching, scraping, and crawling the web.

**Key Tools:**

- `scrape`: Use when you know the exact URL and want structured JSON data.
- `search`: Use to query the web for information.
- `map`: Use to discover URLs on a specific site.
- `crawl`: Use for deep-diving across a whole site (has limits).

When using `scrape`, extracting data as JSON is preferred to avoid context window limits.
