# Brand Design Palette

The visual identity of tupynambalucas.dev uses a refined, modern color triad that reflects developer workspaces, high-end IDE themes, and system stability.

| Color Name       | Hex Code  | Visual Role                | Meaning in Design System                                            |
| :--------------- | :-------- | :------------------------- | :------------------------------------------------------------------ |
| **Deep Indigo**  | `#4C4D9A` | Primary Brand Color        | Repesents system logic, structural depth, and backend stability.    |
| **Royal Blue**   | `#2E55A3` | Secondary Brand Color      | Represents reliability, network integration, and type safety.       |
| **Dusty Violet** | `#AA77A7` | Accent & CTA Color         | Highlights interactive elements, code symbols, and compiler alerts. |
| **Deep Dark**    | `#0F0F0F` | Surface Dark / Background  | Base dark color, mimicking terminal background layouts.             |
| **Light Ice**    | `#F6F6F7` | Surface Light / Background | Base light color for clean print and daylight reading layouts.      |
