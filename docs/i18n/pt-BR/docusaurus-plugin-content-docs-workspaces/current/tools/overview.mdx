O workspace Tools fornece a espinha dorsal de automação para o projeto tupynambalucas.dev. Ele centraliza servidores especializados de Model Context Protocol (MCP) e scripts de infraestrutura projetados para aumentar a produtividade dos desenvolvedores, garantir a paridade de ambiente por meio de compilações Docker multi-stage e permitir fluxos de trabalho de engenharia AI-native seguros e de alto contexto.

Esta seção cobre:

- **Ecossistema MCP**: Servidores de contexto seguros e conteinerizados mapeados através de wrappers Fastify Zero-dependency.
- **Agentes de IA Conteinerizados**: Executando Antigravity e Copilot dentro de sandboxes padrão de containers Docker.
- **Segurança de IA e Rede**: Redes bridge isoladas do Docker e roteamento de gateway HTTP Fastify.
- **Regras de Engajamento**: Convenções comportamentais estritas, diretrizes de código e esquemas de contexto para desenvolvedores de IA.
- **Manutenção e Lógica**: Comandos padronizados de orquestração, blueprints do Compose e manutenção do repositório.
