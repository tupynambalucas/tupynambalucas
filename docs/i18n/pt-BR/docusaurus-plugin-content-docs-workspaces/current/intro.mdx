# Portal de Documentação das Workspaces

Bem-vindo ao **Portal de Documentação das Workspaces do tupynambalucas.dev**. Esta seção fornece detalhamentos técnicos dedicados à arquitetura, design e engenharia interna de cada Contexto Delimitado (Bounded Context) específico dentro do monorepo.

Para decisões arquiteturais de alto nível do projeto e princípios globais, consulte o [Manual de Arquitetura](/docs/explanation/architecture-overview).
