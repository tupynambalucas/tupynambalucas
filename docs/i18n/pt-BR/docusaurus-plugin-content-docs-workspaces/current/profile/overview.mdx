O workspace Profile contém o motor de automação baseado em Node.js e TypeScript que coleta métricas de desenvolvedor do GitHub e as compila em cartões visuais SVG, atualizando dinamicamente o arquivo README.md na raiz do repositório.

---

## Estrutura de Diretórios

```text
profile/
├── dist/             # Arquivos JavaScript compilados e SVGs gerados
├── src/              # Arquivos fonte em TypeScript que implementam o motor de estatísticas
│   ├── clients/      # Clientes de API (GitHub e Fallback de Clone de Git)
│   ├── renderers/    # Compilação dos cartões visuais SVG
│   ├── templates/    # Templates embutidos de SVG e Markdown
│   ├── utils/        # Utilitários de substituição de tokens e de glob matching
│   ├── config.ts     # Carregamento de configurações CLI
│   ├── types.ts      # Interfaces TypeScript e esquemas Zod
│   └── index.ts      # Ponto de entrada coordenando a geração de cartões e compilação do README
├── package.json      # Dependências e scripts do workspace
└── tsconfig.json     # Configuração de compilação do TypeScript e resolução de tipos
```

---

## Operações Escopadas

Execute estes scripts a partir da raiz do monorepo para gerenciar a stack do gerador de perfil:

- `pnpm --filter @tupynambalucas/profile build`: Compila os arquivos TypeScript na pasta `dist/`.
- `pnpm --filter @tupynambalucas/profile dev`: Executa o compilador em modo watch.
- `pnpm --filter @tupynambalucas/profile typecheck`: Valida a integridade dos tipos do workspace.
- `pnpm --filter @tupynambalucas/profile start`: Executa o CLI do gerador de estatísticas compilado.
