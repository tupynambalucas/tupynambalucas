# Padrões de Design e Tamanhos de Logos

Este documento estabelece os padrões de tamanho e dimensões para a aplicação dos logotipos da
marca `tupynambalucas.dev` em aplicações digitais, perfis sociais, mídia impressa e suítes de
software. Para garantir que nossos logotipos mantenham a fidelidade perfeita, desenvolvedores e
designers devem seguir as dimensões padrão listadas abaixo.

Para detalhes sobre o desenvolvimento local no monorepo, consulte a documentação de
[Studio e Ativos Criativos](/workspaces/studio/creative-assets).

---

## 1. Visão Geral das Constantes de Tamanho

Escolher as dimensões corretas do logotipo evita a pixelização, mantém a legibilidade da marca e
garante que os ativos visuais sejam escalados perfeitamente em vários tipos de tela e mídia impressa.

- **Padrão de Resolução de Tela:** 72 DPI (pontos por polegada) é o padrão para exibições web e digitais.
- **Padrão de Resolução de Impressão:** 300 DPI é necessário para uma reprodução nítida e limpa em itens físicos.
- **Formatos Vetoriais (SVG):** Altamente recomendados para interfaces digitais porque escalam
  dinamicamente e são compilados como componentes React dinâmicos.
- **Formatos Rasterizados (PNG):** Preferidos para plataformas digitais que exigem transparência (como
  cabeçalhos de sites ou ícones de perfil de redes sociais).

---

## 2. Tamanhos Padrão de Impressão

Ao exportar logotipos para impressão, formatos vetoriais devem ser usados para evitar perda de
detalhes. Abaixo estão as dimensões recomendadas e os requisitos de DPI:

| Collateral / Use Case  | Recommended Dimensions (inches) | DPI Target | Layout & Placement Notes                           |
| :--------------------- | :------------------------------ | :--------- | :------------------------------------------------- |
| **Business Cards**     | 1.5″ × 1.5″                     | 300 DPI    | Typically square; positioned top-left or centered. |
| **Letterhead**         | 2″ × 2″ or smaller              | 300 DPI    | Subtle scaling (A4 size: 210mm x 297mm).           |
| **Brochures & Flyers** | 2.0″ to 3.0″ wide               | 300 DPI    | Scales depending on the overall layout.            |
| **Posters & Banners**  | 5.0″ to 12.0″ wide (or larger)  | 300 DPI    | Adjust dynamically to match the banner canvas.     |
| **T-Shirts/Merch**     | 8″ × 10″ (centered)             | 300 DPI    | High-resolution vector preferred.                  |
| **Packaging/Labels**   | 1″ to 3″ wide                   | 300 DPI    | Adjust based on label or package area.             |

### Contexto Adicional de Impressão

- **Cartões de Visita (Tamanho do Cartão):** 3.5 in x 2.0 in (o logotipo em si costuma ter 0.75 in x
  0.75 in).
- **Canecas (Padrão 11 oz):** 7.5 in x 3.0 in.
- **Bonés:** 3.0 in a 3.5 in de largura.
- **Sacolas Ecológicas (Tote Bags):** 14.0 in x 16.0 in.
- **Posicionamento em Camisetas:** Peito de adultos (3.5 in x 3.5 in), peito de crianças (3.3 in x
  3.3 in), área máxima de impressão (14.0 in x 15.0 in) e logotipos de bolso (4.25 in x 5.25 in).
- **Envelopes (Padrão/C4):** 22.9 cm x 32.4 cm (tamanho C4) ou 4.125 in x 9.5 in.

## 3. Tamanhos para Web e Digital

Para canais digitais, mantenha o tamanho dos arquivos minimizado (por exemplo, abaixo de 30 KB para
assinaturas de e-mail) para reduzir o tempo de carregamento da página.

| Digital Use Case        | Recommended Dimensions (pixels) | Aspect Ratio  | Format Recommendations                             |
| :---------------------- | :------------------------------ | :------------ | :------------------------------------------------- |
| **Website Header Logo** | 250 × 100 px (average)          | Flexible      | PNG (for transparency) or SVG.                     |
| **Favicon**             | 16 × 16 px or 32 × 32 px        | 1:1           | ICO or transparent PNG.                            |
| **Email Signature**     | 320 × 100 px                    | 3.2:1         | Keep under 30 KB (typically 300-400px x 70-100px). |
| **Mobile App Icon**     | 512 × 512 px                    | 1:1           | PNG format preferred for application stores.       |
| **Mobile App Logo**     | 864 × 864 px (design)           | 1:1           | Canvas size used for high-fidelity assets.         |
| **Desktop App Icon**    | 256 × 256 px                    | 1:1           | Vector format recommended for high-res screens.    |
| **Banner Logo**         | 1800 × 200 px                   | 9:1           | Horizontally oriented banner graphics.             |
| **Product Packaging**   | 1920 × 1020 px                  | 16:9 (approx) | High-definition digital previews.                  |

### Especificidades de Plataformas de CMS e E-Commerce

- **Cabeçalhos do WordPress:** Logotipos horizontais devem visar 250 x 45 px ou 250 x 55 px.
- **Temas do Shopify:** O tema Parallax, por exemplo, recomenda 410 x 205 px.
- **Modelos do Squarespace:** O modelo do tema Bedford exige uma altura de logotipo de 100 px.

## 4. Tamanhos para Redes Sociais

As redes sociais comprimem os arquivos enviados. O uso de formatos PNG de alta qualidade (ou JPGs
para o Instagram) produzirá o melhor resultado visual.

| Social Platform | Profile Picture Dimensions (px) | Cover / Banner Dimensions (px) | Notes & File Types                            |
| :-------------- | :------------------------------ | :----------------------------- | :-------------------------------------------- |
| **Facebook**    | 170 × 170                       | 820 × 312                      | Use PNG files for high quality.               |
| **Instagram**   | 320 × 320                       | N/A                            | Instagram converts all assets to JPG.         |
| **Twitter (X)** | 400 × 400                       | 1500 × 500                     | Max file size: 5 MB (PNG, JPG, or GIF).       |
| **LinkedIn**    | 400 × 400                       | 1128 × 191                     | Max file size: 10 MB (PNG, JPG, or GIF).      |
| **YouTube**     | 800 × 800                       | 2560 × 1440                    | Desktop/TV banner sizes (PNG, JPG, GIF, BMP). |
| **Twitch**      | 1600 × 1600                     | N/A                            | High-resolution square profiles.              |
| **Discord**     | 128 × 128                       | N/A                            | Keep to 1:1 aspect ratio; max size 8 MB.      |
| **Pinterest**   | 165 × 165                       | 800 × 450                      | Profile and showcase boards.                  |
| **TikTok**      | 200 × 200                       | N/A                            | Circle-cropped profile pictures.              |

## 5. Recomendações de Tela para Software de Design

Ao iniciar novos projetos de logotipo em suítes de edição profissional, estabeleça os seguintes
tamanhos de tela iniciais:

### Adobe Photoshop

- **Tela Inicial Padrão:** 1000 × 600 px ou 500 × 500 px.
- **Tela Escalonável:** 1200 × 1200 px (permite redimensionamento fácil sem distorção visual).

### Adobe Illustrator

- **Tela Padrão:** 320 × 500 px (otimizada para proporções de tela de dispositivos móveis).
