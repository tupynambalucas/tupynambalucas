# Diretrizes Criativas

Esta seção consolida as diretrizes visuais, tokens de design e padrões de ativos digitais e
impressos da identidade de marca do tupynambalucas.dev. Ela serve como a única fonte de verdade
para designers e engenheiros frontend que trabalham com componentes visuais, logotipos e layouts.

---

## 1. Princípios Fundamentais

Nosso sistema criativo é projetado para transmitir precisão, alta performance e excelência visual.
Ele garante consistência em:

- **Interfaces Digitais:** SPAs React responsivos, camadas de renderização WebGL/Three.js e favicons.
- **Redes Sociais:** Banners, capas e fotos de perfil para redes de desenvolvedores.
- **Entregáveis Físicos:** Cartões de visita, papelaria e mercadorias da marca.
