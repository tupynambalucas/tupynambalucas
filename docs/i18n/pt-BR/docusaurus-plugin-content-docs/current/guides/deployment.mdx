# Como Implantar

Este guia descreve as sequências de comandos de implantação para publicar o tupynambalucas.dev.

As implantações são executadas diretamente na máquina hospedeira usando nossos scripts de CLI de raiz unificados.

Consulte a [Referência de Comandos](../reference/commands.mdx) para mais cenários de inicialização/parada.
